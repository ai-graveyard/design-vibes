Section 02 / Doctrine

House rules

  1. HTML is a design system. It shipped in 1993 and the docs are one page.
  2. If it loads fast, it looks good. This is the entire theory of aesthetics.
  3. Underlines belong to links. Everything else must earn its ink.
  4. Every byte is a guest in someone’s pocket. Act like a host.
  5. When in doubt, make the heading bigger. When certain, bigger still.
