Undo is a whole word

Every rev is immutable and kept warm for thirty days. undo repoints the edge at the previous build — no rebuild, no ticket, no confirmation theater.

$ shipd undo
[ok] rev 4b1e77d restored in 0.6s
[ok] traffic settled · p99 39ms
cmd.02preview

A URL per branch

preview builds your working tree and hangs it on a throwaway subdomain wired to real infra. It sleeps when idle and vanishes on merge.

$ shipd preview
[ok] built feat-billing in 3.1s
[ok] up at pv-83ax.shipd.dev

Logs that keep up

One stream across every region — structured, greppable, timestamped in your zone. Follow mode survives redeploys, so your pipe never dies mid-incident.

# man 1 shipd — the whole interface
SHIPD(1)General Commands ManualSHIPD(1)

Name

shipd — ship code from a shell to production, and prove it happened

Synopsis

shipd <command> [options] shipd deploy [-p | --prod] [--stage] [--region <id>]... [-q] [--dry-run] shipd undo | status | logs [-f] | preview | env <get|set>

Description

shipd reads shipd.toml when present and infers honest defaults when it is not. Builds are hermetic and content-addressed: the same tree always produces the same rev, and a rev that built once is never built again. Nothing leaves your machine except the artifact — signed, compressed, and accounted for.

Options

-p, --prod
Deploy to production. Requires a clean tree, a reachable origin, and the flag spelled out — no alias, on purpose.
--stage
Same pipeline, disposable consequences. Staging revs expire after seven days.
--region <id>
Constrain the deploy to one or more regions (fra1, sfo2, sin1, …). Repeatable. Default: all nine.
-q, --quiet
Print errors only. The exit status is the whole story.
--dry-run
Build, bundle, plan, price it out — everything except the consequences.

Exit Status

0deployed; traffic fully shifted 1build or bundle failed; nothing left your machine 2policy refused the deploy (dirty tree, failing canary) 4canary failed; previous rev restored automatically

See Also

shipd-undo(1), shipd-env(1), tar(1) — for how it used to feel.
